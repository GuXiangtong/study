### 切入点

已知等式中同时有cosα和sinα，而二倍角公式中sin2α = 2sinαcosα，cos2α = cos²α - sin²α。你能想到什么方法把cosα - sinα = 1和sin2α联系起来？提示：试试两边平方。

### 路径推演

第一步：将cosα - sinα = 1两边平方，得到(cosα - sinα)² = 1，即cos²α - 2sinαcosα + sin²α = 1。第二步：利用cos²α + sin²α = 1，化简得1 - sin2α = 1，所以sin2α = 0。第三步：由sin2α = 0，得到cos2α = ±1。第四步：如何确定cos2α的符号？回到原式cosα - sinα = 1，因为cosα - sinα = √2cos(α + π/4) = 1，所以cos(α + π/4) = √2/2，说明α + π/4 = 2kπ ± π/4，分析α的范围即可得到cos2α = 1。第五步：代入公式cos(2α + π/3) = cos2α·cos(π/3) - sin2α·sin(π/3) = 1×(1/2) - 0×(√3/2) = 1/2。

### 易错提醒

常见错误：① 平方后忘记检验符号，直接取cos2α = ±1，导致答案多一个-1/2；② 计算sin2α时混淆成sin²α；③ 两角和的余弦公式中符号记反（应该是cos(A+B)=cosAcosB - sinAsinB）。

### 验证方法

将结果代回原式验证：若cos(2α + π/3) = 1/2，且sin2α = 0，则cos2α = 1。由cosα - sinα = 1可解得α = 2kπ，代入验证成立。如果取了cos2α = -1，会导致α = 2kπ + π/2，此时cosα - sinα = -1，与条件矛盾。

### 提高题

已知 $\cos\alpha - \sin\alpha = \frac{1}{2}$，且 $\alpha \in (0, \frac{\pi}{4})$，求 $\cos2\alpha$ 的值。

解题思路：两边平方得 $1 - \sin2\alpha = \frac{1}{4}$，所以 $\sin2\alpha = \frac{3}{4}$。由 $\alpha \in (0, \frac{\pi}{4})$ 得 $2\alpha \in (0, \frac{\pi}{2})$，故 $\cos2\alpha = \sqrt{1 - \sin^2 2\alpha} = \sqrt{1 - \frac{9}{16}} = \frac{\sqrt{7}}{4}$。

参考答案：$\cos2\alpha = \frac{\sqrt{7}}{4}$

### 难题
