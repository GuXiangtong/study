## 第三步：引导思考，给出解题路径

### 审题

题目给了我们一个关于α的等式，要求的是另一个关于2α的三角函数值。请想一想，我们如何把条件里的α和问题里的2α联系起来？常用的桥梁是什么？

### 切入点

已知条件是一个关于 cosα 和 sinα 的线性方程，我们还有一个隐藏的方程，你是否能想到是什么？对了，就是 sin²α + cos²α = 1。有了这两个方程，我们能否求出 cosα 和 sinα 的具体值？还是说可以求出它们的某种组合，比如 cosα·sinα？

### 路径推演

思路一：联立方程法。将 cosα - sinα = 1 与 sin²α + cos²α = 1 联立。你能通过代入法求出 sinα 和 cosα 的值吗？注意，解出的值可能有多种情况，但目标表达式 cos(2α + π/3) 的值是否唯一？思路二：平方法。将已知等式两边平方，得到 (cosα - sinα)² = 1。展开后，利用平方关系，你能得到什么？是不是可以求出 sin2α 的值？求出 sin2α 后，要得到 cos(2α + π/3)，我们还需要什么？是不是还需要 cos2α？那么 cos2α 怎么求？别忘了 sin²2α + cos²2α = 1，但要注意符号！cos2α 的正负由什么决定？

### 易错提醒

这类题最常见的错误是：1) 联立方程求解时，解出 sinα 和 cosα 的值后，忘记检验它们是否同时满足原始方程。2) 使用平方法时，忽略了平方可能引入增根，需要对 cosα - sinα = 1 的条件进行验证，以确定 sin2α 或 cos2α 的符号。3) 在计算 cos2α 时，直接套用公式 cos2α = 1 - 2sin²α 或 2cos²α - 1，但代入的是错误的 sinα 或 cosα 值。4) 对两角和的余弦公式 cos(2α + π/3) 展开时，符号搞反。

### 验证方法

将你求出的 sinα 和 cosα 的值代回原条件 cosα - sinα = 1 中，看等式是否成立。同时，由于三角函数的多值性，你可以考虑辅助角公式，将已知条件化为 √2 cos(α + π/4) = 1，从而直接求出 cos(α + π/4) 的值，然后利用二倍角公式求出 cos(2α + π/2) 的值，最后再展开成 cos(2α + π/3)。这个结果应该和你之前算出的结果一致，可以互相验证。

## 第四步：生成巩固练习

针对薄弱知识点生成三道变式练习题：

### 基础题

已知 $\sin\alpha = \frac{3}{5}$，且 $\alpha$ 为第二象限角，求 $\cos(\alpha + \frac{\pi}{3})$ 的值。

解题思路：1. 由 $\sin\alpha = \frac{3}{5}$ 和 $\alpha$ 在第二象限，得 $\cos\alpha = -\frac{4}{5}$。
2. 利用两角和的余弦公式：$\cos(\alpha + \frac{\pi}{3}) = \cos\alpha\cos\frac{\pi}{3} - \sin\alpha\sin\frac{\pi}{3} = (-\frac{4}{5})\cdot\frac{1}{2} - \frac{3}{5}\cdot\frac{\sqrt{3}}{2} = \frac{4-3\sqrt{3}}{10}$。

参考答案：$\frac{4-3\sqrt{3}}{10}$

### 提高题

已知 $\sin\alpha + \cos\alpha = \frac{1}{2}$，求 $\sin2\alpha$ 的值。

解题思路：1. 将已知等式两边平方：$(\sin\alpha + \cos\alpha)^2 = (\frac{1}{2})^2$。
2. 展开得：$\sin^2\alpha + 2\sin\alpha\cos\alpha + \cos^2\alpha = \frac{1}{4}$。
3. 利用平方关系 $\sin^2\alpha + \cos^2\alpha = 1$ 和二倍角公式 $\sin2\alpha = 2\sin\alpha\cos\alpha$，代入得：$1 + \sin2\alpha = \frac{1}{4}$。
4. 解得 $\sin2\alpha = -\frac{3}{4}$。

参考答案：$-\frac{3}{4}$

### 难题

已知 $\cos(\alpha - \frac{\pi}{6}) = \frac{3}{5}$，且 $\alpha \in (0, \frac{\pi}{2})$，求 $\sin(2\alpha + \frac{\pi}{3})$ 的值。

解题思路：1. 令 $\beta = \alpha - \frac{\pi}{6}$，则 $\cos\beta = \frac{3}{5}$，且由 $\alpha$ 范围得 $\beta \in (-\frac{\pi}{6}, \frac{\pi}{3})$，故 $\sin\beta = \frac{4}{5}$。
2. 目标角 $2\alpha + \frac{\pi}{3} = 2(\alpha + \frac{\pi}{6}) = 2(\beta + \frac{\pi}{3}) = 2\beta + \frac{2\pi}{3}$。
3. 先求 $\sin2\beta = 2\sin\beta\cos\beta = 2\cdot\frac{4}{5}\cdot\frac{3}{5} = \frac{24}{25}$，$\cos2\beta = 2\cos^2\beta - 1 = 2\cdot\frac{9}{25} - 1 = -\frac{7}{25}$。
4. 则 $\sin(2\alpha + \frac{\pi}{3}) = \sin(2\beta + \frac{2\pi}{3}) = \sin2\beta\cos\frac{2\pi}{3} + \cos2\beta\sin\frac{2\pi}{3} = \frac{24}{25}\cdot(-\frac{1}{2}) + (-\frac{7}{25})\cdot\frac{\sqrt{3}}{2} = -\frac{24+7\sqrt{3}}{50}$。

参考答案：$\frac{24}{25}$
