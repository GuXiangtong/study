针对薄弱知识点生成三道变式练习题：

### 基础题

已知函数 f(x) 的导函数为 f'(x)=x^2。若点 (1, y0) 在导函数图像上，求 y0 的值，并判断 1 是否为函数 f(x) 的「T类数」（定义同上题）。

解题思路：由点 (1, y0) 在 f'(x)=x^2 图像上，得 y0 = f'(1) = 1^2 = 1。
计算 f'(1) · f'(y0) = 1^2 · f'(1) = 1 × 1^2 = 1 ≥ 0，满足定义，故 1 是T类数。

参考答案：y0 = 1，1 是T类数。

### 提高题

已知定义域为 R 的函数 y=f(x) 的导函数 f'(x) 连续，且 f'(x) 的值域为 (0, +∞)。集合 S 为函数 f(x) 的「T类集」（定义同上题第(2)问）。求证：S = R。

解题思路：1. 由 f'(x) 值域为 (0, +∞) 知，对任意 x ∈ R，f'(x) > 0。
2. 任取 x0 ∈ R，记 y0 = f'(x0)，则 y0 > 0。
3. 因为 y0 也是实数，所以 f'(y0) > 0。
4. 于是 f'(x0) · f'(y0) > 0，满足「T类数」条件，x0 ∈ S。
5. 由 x0 的任意性，S = R。

参考答案：证明：任取 x0 ∈ R，y0 = f'(x0) > 0。因为 f'(x) 的值域为 (0, +∞)，所以 f'(y0) > 0。从而 f'(x0)·f'(y0) > 0 ≥ 0，故任意 x0 都是T类数，S = R。

### 难题

已知函数 f(x) = (1/3)x^3 + ax，a 为实常数，其导函数为 f'(x)。记函数 f(x) 的「T类集」为 S（定义：若点 (x0, y0) 在 f'(x) 图像上，且 f'(x0)·f'(y0) ≥ 0，则 x0 ∈ S）。若 S = R，求实数 a 的取值范围。

解题思路：f'(x) = x^2 + a。
T类集为 R，等价于对任意 x0 ∈ R，取 y0 = f'(x0) = x0^2 + a，有 f'(x0)·f'(y0) = (x0^2+a) · (y0^2+a) ≥ 0。
即 (x0^2+a) · [(x0^2+a)^2 + a] ≥ 0 对一切 x0 ∈ R 恒成立。
记 t = x0^2 + a，则 t ≥ a。不等式化为 t · (t^2 + a) ≥ 0 对 t ∈ [a, +∞) 恒成立。
分析：
1. 若 a ≥ 0，则 t ≥ a ≥ 0，t^2+a > 0，故 t(t^2+a) ≥ 0 恒成立。
2. 若 a < 0，则当 t ∈ [a, 0) 时，t < 0。要使 t(t^2+a) ≥ 0，需 t^2+a ≤ 0。但 t^2+a 在 t ∈ [a, 0) 上，当 t 接近 0 时，t^2+a 接近 a < 0，可以满足；但当 t 在 a 附近时，t^2+a 可能为正。实际上，取 t = a，则 a(a^2+a) = a^2(a+1)。由于 a < 0，当 a < -1 时，a+1 < 0，乘积 a^2(a+1) < 0，不成立。因此 a < 0 时无法保证恒成立。
综上，a 的取值范围是 [0, +∞)。

参考答案：a ≥ 0
